# SnapStrip Studio - Free Online Photo Booth 📸

## 🎉 [**TRY IT NOW - LIVE DEMO**](https://thanhnguyn.github.io/Photo-Booth/)

**SnapStrip Studio** is a completely free online photo booth application. Create stunning photo strips with 12 creative templates in just a few clicks!

---

## ✨ Use It Now - No Installation Required!

### 🌐 Access Directly At:
**👉 [https://thanhnguyn.github.io/Photo-Booth/](https://thanhnguyn.github.io/Photo-Booth/)**

### 📱 How to Use (5 Simple Steps):

1. **Visit the website** at the link above
2. **Click START** on the homepage
3. **Choose Layout** - How many poses do you want? (2, 3, 4, or 6 photos)
4. **Select Template** - 12 creative concepts to choose from:
   - 📰 Newspaper, 📸 Magazine, 💥 Comic
   - 📷 Polaroid, 🎬 Movie, ⚡ Neon
   - And many more...
5. **Choose Photo Source**:
   - **📷 Use Camera** - Take photos directly from webcam (camera permission required)
   - **📤 Upload Photos** - Upload existing photos from your computer/phone

**🎊 Done! Download your photo strip!**

## 🚀 Development & Deployment

### Local Development:

**Option 1: Direct Open**
```bash
cd docs
# Double click index.html in your file explorer
```

**Option 2: Live Server (VS Code)**
```bash
# Install "Live Server" extension in VS Code
# Right-click on docs/index.html → Open with Live Server
```

**Option 3: Python HTTP Server**
```bash
cd docs
python -m http.server 8000
# Open http://localhost:8000 in browser
```

**Option 4: Node.js HTTP Server**
```bash
npx http-server docs -p 8000
# Open http://localhost:8000 in browser
```

---

### Deploy to GitHub Pages:

See detailed instructions in [DEPLOYMENT.md](DEPLOYMENT.md)

**Quick Deploy Steps:**

1. **Push code to GitHub:**
```bash
git add .
git commit -m "Your commit message"
git push origin main
```

2. **Enable GitHub Pages:**
   - Go to repository Settings → Pages
   - Source: `main` branch, `/docs` folder
   - Click Save
   - Wait 1-2 minutes

3. **Access your live site:**
   - URL: `https://YOUR-USERNAME.github.io/Photo-Booth/`
   - HTTPS enabled automatically ✅
   - Camera will work! 📷

## 📝 Frequently Asked Questions

**Q: Is this really free?**  
A: Yes! 100% free, no hidden costs, no watermarks, no premium features.

**Q: Are my photos uploaded to a server?**  
A: NO! All processing happens in your browser. Privacy first!

**Q: I don't have a webcam, can I still use it?**  
A: Yes! Choose "Upload Photos" instead of "Use Camera".

**Q: Does it work on mobile phones?**  
A: Yes! The website is fully responsive and works on all devices.

**Q: Can I print the photo strips?**  
A: Yes! Download as JPG and print normally.

**Q: How do I share my photo strips?**  
A: Click the Share button or download and post to social media.

**Q: Can I use this for commercial events?**  
A: Yes! Feel free to use it for any purpose.

**Q: Is internet required?**  
A: Only for initial load (to get Tailwind CSS). Upload mode can work offline after that.

**Q: Can I customize the templates?**  
A: Currently, you can choose from 12 pre-made templates. Custom templates coming soon!

**Q: What image formats are supported for upload?**  
A: JPG, PNG, and most common image formats.
